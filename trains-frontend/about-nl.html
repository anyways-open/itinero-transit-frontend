Over dit project

Deze routeplanner is ontwikkeld door het OpenPlannerTeam-project met als doel het bouwen van de volgende generaties route planners. De basis features zijn:

  • Publiceren van data volgens Linked Data-principes.
  • Een snelle en flexibele route planner die zowel client als server zijn kan runnen.
  • Een Open-Source licentie.
Meer weten of wil je dit hergebruiken, dan start je best bij de documentatie:
  • Itinero documenatie (EN)
  • Itinero.Transit HTTP API documentation (EN)
